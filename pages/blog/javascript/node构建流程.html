# node构建

涉及到的语言

  • javascript
  • python
  • make
  • c++

# ./configure 准备

  • ./tools/gyp_node.py -f make

    • ./configure 准备 GYP 执行需要的参数
    • 然后调用 gyp_node 中的 run_gyp(args)
    • 生成 icu_config.gypi 、 ./config.gypi 、 ./config.mk
    • 主要目的是生成了一些辅助构建 make 和 gyp 的引用文件，生成 out 目录，将配置进行预编译

  • 读取 node.gyp

    • targets
      • node 核心 node_js2c
      • cctest
    • variable
    • condition

  • 生成 ./out/*.mk

# make 编译

  • 读取 Makefile
  • include 上一步生成的 ./out/*.mk
  • 编译依赖，测试，性能测试
